California Housing test podataka¶

Set podataka california houses prices sastoji se od $8$ znaajki:

  1. medianIncome ("MedInc") - Prosjena zarada domainstava koja se nalaze u okolici (mjereno u 10k),

  2. housingMedianAge ("HouseAge) - Prosjena starost kua koje se nalaze u okolici (nii broj novija zgrada),

  3. averageRooms ("AveRooms") - Prosjean broj soba u kuama koje se nalaze u okolici,

  4. averageBedrooms ("AveBedrms") - Prosjean broj spavaih soba u kuama koje se nalaze u okolici,

  5. population ("Population") - Prosjean broj ljudi koji ive u okolici,

  6. households ("AveOccup") - Prosjean broj ljudi po domainstvu,

  7. latitude ("Latitude") - Mjera koja kae koliko je sjeverno kua,

  8. longitude ("Longitude") - Mjera koja kae kolo je zapadno kua.

Dakle temeljem ovih 8 znaajki i cijene koliko pojedina kua vrijedi, potrebno je kreirati takav model koji e predvidjeti cijenu neke nove kue ako su za nju poznate prethodno navedene znaajke.

Vie o podacima moe se pronai na poveznicama: docs, sklearn.

Klasa za testiranje algoritama: model¶

U svrhu lakeg kreiranja neuronskih mrea i testiranja algoritama stvorena je klasa model koja slui kao svojevrsni API nad TensorFlow knjiicom. Cijela neuronska mrea zadaje se pomou $9$ metoda:

  1. setLayersNeurons - prima listu koja ima definiran broj neurona po sloju poevi s prvim skrivenim slojem. Primjerice neuronska mrea koja ima topologiju 500,200,200 i izlaz 1 zadala bi se pomou setLayersNeurons([500,200,200,1]).

  2. setActivations - prima listu koja ima definirane aktivacijske funkcije. Primjerice ako elimo da prva dva sloja imaju Relu kao aktivacijsku funkciju, a zadnja dva da budu bez aktivacije, poziv metode je sljedei: setActivations(['relu','relu',None,None]) Implementirane aktivacijske funkcije jesu:

    • 'relu' - Rectified Linear Unit).
    • 'elu' - Exponential linear unit.
    • 'sig' - Sigmoid.
    • 'tanh' - Hyperbolic Tangent Activation Function.
    • None - Kada ne elimo imati aktivacijsku funkciju nad slojem.

  3. setRugluarizations - Metoda koja dodaje svakom sloju regularizaciju. Implementirane su dvije vrste regularizacije: l1 i l2 regularizacija. Uz svaku regularizaciju potrebno je dodati i regularizacijski parametar, stoga koritenje ove funkcije jest sljedee: setRegularization([['l2',0.001],['l2',0.001],['l1',0.01],None]) gdje se na prva dva sloja postavlja regularizacija l2 s regularizacijskim parametrom $0.001$, na trei sloj se postavlja regularizacije l1 s regularizacijskim parametrom $0.01$, dok etvrti sloj nema regularizacije.

  4. setBatchNormalization - Metoda koja primjenjuje Batch Normalization na slojeve, kao argument prima listu vrijednosti momentuma koji se koristi prilikom auriranja batch normalization matrice (None ukoliko nema normalizacije). Primjer pozivanja funkcije glasi: setBatchNormalization([0.9, 0.9, 0.9, None])

  5. setInitialisation - Metoda koja postavlja inicijalne vrijednosti teina koje se treniraju. Implementirana je He inicijalizacija. Koritenje jest vrlo jednostavno: setInitialisation(['he_init','he_init','he_init',None]). U sluaju kada se koristi None, teine su nasumino inicijalizirane.

  6. setLoss - Metoda koja postavlja loss funkciju. Loss funkcija moe biti: softmax ili mse, odnosno softmax za klasifikaciju ili srednja kvadratna pogreka za regresiju. Koritenje metode je sljedee: setLoss("softmax") i setLoss("mse").

  7. setOptimiser - Metoda koja postavlja optimizator. Ugraeni optimizatori su:

    • Adam - setOptimiser(["adam", $\alpha$]=
    • AdaGrad - setOptimiser(["adaGrad", $\alpha$])
    • Gradient Descent - setOptimiser(["gradDes", $\alpha$])
    • Momentum - setOptimiser(["momentum", $\alpha$, $\beta$])
    • RMSProp - setOptimiser(["rmsProp", $\alpha$, $\beta$])

  8. setIoShape - Metoda koja postavlja oblik ulaznih i izlaznih podataka kao i njihov tip. Za broj stupaca se tipino postavlja None. Metoda se poziva tako da se prvo navodi oblik ulaznog sloja, potom oblik izlaza, a zatim tipovi ulaznih i izlaznih podataka. Primjerice: setIOshape((None,9),(None), tf.float32, tf.int32) postavlja ulazni sloj koji e imati $9$ znaajki tipa float, izlazni sloj e imati $1$ znaajku i bit e tipa integer.

  9. setAlphaDecay - Metoda omoguuje dodavanje raspada parametra $\alpha$. Poziva se na sljedei nain: setAlphaDecay($\alpha_0$,$r$,$baza$) (tipina baza je $1/10$).

  10. trainModel - Metoda koja trenira model. Kao argumente prima redom: ulazni parametri trening seta, izlaz trening seta, ulazni parametri validacijskog seta, izlaz validacijskog seta, broj epoha, veliina batch-a, early stopping kriterij nakon kojeg se zaustavlja treniranje.

Testirani modeli¶

Svi modeli su imali iste parametre:

  1. Broj neurona i slojeva: 50, 50, 25, 10, 1,
  2. Aktivacije: relu, relu, relu, relu, None,
  3. Regularizacija: ['l2',0.001],['l2',0.001], ['l2',0.001],['l2',0.001],None,
  4. BatchNormalization: 0.9, 0.9, 0.9, 0.9, 0.9
  5. Inicijalizacija: He inicijalizacija, He inicijalizacija, He inicijalizacija, He inicijalizacija, He inicijalizacija
  6. Loss funkcija: MSE
  7. Ulazni sloj/izlazni sloj: 9 parametara tipa float/ 1 (vrijednost kue) tipa float
  8. Trenirali su se s ukljuenim early stop algoritmom na 10000 epoha s veliinom batch-a 200 i early stopping kriterijem nakon 25 nepromijenjenih najboljih rezultata.
  9. Gradient descent i Momentum optimizator trenirali su se i sa AlphaDecay-om (parametri: 0.001,10000,1/10)
  10. Inicijalni $\alpha = 0.001$

        for batch_idx in np.array_split(rnd_idx, n_batches):
            X_batch, y_batch = X[batch_idx], y[batch_idx]

        for i, elem in enumerate(self.activations):

                else:
                    learning_rate = self.optimiser[1]
                optimizer = tf.train.AdagradOptimizer(learning_rate)
            
            if (self.optimiser[0] == "adam"):

import matplotlib.pyplot as plt
import numpy as np

def plot_pareto_frontier(Xs, Ys, maxX=True, maxY=True):
    '''Pareto frontier selection process'''
    sorted_list = sorted([[Xs[i], Ys[i]] for i in range(len(Xs))], reverse=maxX)
    pareto_front = [sorted_list[0]]
    for pair in sorted_list[1:]:
        if maxY:
            if pair[1] >= pareto_front[-1][1]:
                pareto_front.append(pair)
        else:
            if pair[1] <= pareto_front[-1][1]:
                pareto_front.append(pair)
    
    '''Plotting process'''
    plt.scatter(Xs,Ys, color="red", marker="x")
    pf_X = [pair[0] for pair in pareto_front]
    pf_Y = [pair[1] for pair in pareto_front]
    plt.plot(pf_X, pf_Y, color = "blue")
    plt.xlabel("Epoch")
    plt.ylabel("Score")
    plt.title("Pareto fronta")
    plt.show()
    return(pf_X,pf_Y)
